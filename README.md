# 🛡️ Валидатор транзакций с Pydantic(Pydantic + Pytest)
Микросервис для строгой валидации финансовых транзакций. Проект фокусируется на защите системы от некорректных данных и математической проверке банковских карт.

## 🛠 Стек технологий

* Framework: FastAPI
* Validation: Pydantic v2 (использование Field, Enum, Optional)
* Testing: Pytest
* Logic: Алгоритм Луна (Luhn Algorithm) реализован с использованием List Comprehensions и Generators.

## 🧩 Ключевые особенности

* Пакетная обработка: Эндпоинт принимает список транзакций и фильтрует их «на лету» с помощью генераторов, экономя память.
* Валидация: Номера карт проверяются математически на контрольную сумму.
* Type Safety: Использование Enum для типов транзакций (debit/credit) исключает логические ошибки.

## 🚀 Запуск и тестирование

   1. Запуск сервера:
   
   uvicorn app.main:app --reload
   
   2. Запуск автоматических тестов:
   
   pytest
   
   
## 📈 Пример работы API
Эндпоинт: POST /validate
Вход (Request Body):

[
  {
    "amount": 100.0,
    "type": "credit",
    "card_number": "4539148408597218"
  },
  {
    "amount": 50.0,
    "type": "debit",
    "card_number": "4539148408597219" 
  }
]

Выход (Response):
Система вернет только первую транзакцию, так как вторая не пройдет проверку алгоритмом Луна.

{
  "status": "success",
  "valid_count": 1,
  "total_processed": 2,
  "data": [
		{
			"amount": 100.0,
			"type": "credit",
			"card_number": "4539148408597218"
  	}
	]
}
